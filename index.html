Caboodle Tech Application Branding

This application allows users to easily create the branding for a Caboodle Tech Inc module or application. Badges for Caboodle Tech Inc modules and applications have a unique identity that include a mono-space font (Roboto) for all text within the badge and a periodic table inspired square cell layout. The square badge has a thick border surrounding it. Inside the badge in the upper left and right are date markers comprised of a letter and four digit year, where the letter preceding the year is always an uppercase letter in the range A - L representing the month respectively; A = January ... L = December. The upper left date is when the module or application was first introduced. The upper right is when the module or application was last updated. Centered in the upper middle of the badge is a two or three letter acronym representing the module or applications full name. Only the first letter of this acronym is uppercase following the style most common on periodic tables. The lower middle of the badge is the applications full or abbreviated name cut down to two lines of text maximum. Centered at the bottom of the badge is the module or applications current version in one of three formats. Valid version values are "Alpha: " followed by a whole number, "Beta: " followed by a whole number, or a semantic versioning number in the pattern of X.Y.Z where X, Y, and Z are non-negative integers that do not contain leading zeroes. X is the major version, Y is the minor version, and Z is the patch version; each element increases numerically for each module or application individually.

Instructions for Use

To exclude a word from being used to form the module or application acronym place it inside brackets. For example "Caboodle [Tech] App Branding" would ignore the word Tech and create the acronym of Cab.

Any version number with the pattern 0.0.# will be considered an Alpha release, a pattern of 0.#.0 will be considered a Beta release, and the pattern of 1.0.0+ will be considered a production release.
